# webdev_project
*Виконала Сачек Владислава КА-31*   
Мета веб-сайту: надати користувачу можливість переглядати опис книг, фільмів й серіалів, відкривати сторінки творів, виконувати базовий пошук та ознайомлюватися з контентом сервісу.   
   
Список створених сторінок:
1. Головна сторінка (Home)
2. Каталог (Catalog)
3. Сторінки окремих творів (Interstellar, Monster, Never Let Me Go)
4. Про сайт (About)
5. Контакти (Contacts)
6. Авторизація (Login)

На всіх сторінках використано стандартну структуру HTML5, семантичні теги, заголовки, абзаци, створене навігаційне меню. Також використані списки: неупорядковані ul для списків творів у каталозі, рекомендація та для меню навігації; упорядковані ol у breadcrumbs на сторінках окремих творів, щоб показати послідовні переходи. Створені форми для пошуку, входу і реєстрації, зворотного зв'язку з використанням атрибутів валідації. Для фільтрів у каталозі були використані випадаючі списки <select>, для введення року та вибору жанрів – поле введення <input>, з типами number та checkbox відповідно.
